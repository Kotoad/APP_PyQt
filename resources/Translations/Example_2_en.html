Goal is to teach you how to use conditional logic in your diagrams.

You will need:

  • Any Raspberry Pi board
  • USB cable to connect the board to your computer
  • Any LED
  • 220 Ohm resistor
  • Push button
  • Breadboard and jumper wires

How to make it work:

The flowchart will consists of the following elements:

  1. 1x Start: Initializes the program.
  2. 2x Switch: Controls the LED state (ON/OFF).
  3. 1x If: Checks the button state.
  4. 1x While: Creates an infinite loop.
  5. 1x End: Terminates the program.

Flowchart Logic:

  1. The program starts with the Start element.
  2. It then enters a loop where it checks the state of the button using the If element.
  3. If the button is pressed, it turns the LED ON using the first Switch element.
  4. If the button is not pressed, it turns the LED OFF using the second Switch element.
  5. This loop continues indefinitely until the program is manually stopped or terminated using the End element.

Creating the Flowchart:

  1. Create new varible by clicking the "Variables" button in the main window toolbar.
    Name it however u want and set its type to Boolean.
  2. Create new device by clicking the "Devices" button in the main window toolbar.
    Name it LED for convenience and set its type to Output.
  4. Create another device named Button and set its type to Input.
  6. Then add the required elements to the canvas as described in the previous example.
  7. Connect the elements in the following order:
    • "Start → While", make sure to set the condition to True to create an infinite loop.
    • "While (Connect to top connection point) → If"
    • "While (Connect to bottom connection point) → End"
    • "If (True) → Switch (LED ON)"
    • "If (False) → Switch (LED OFF)"
    • "Switch (LED ON) → End"
    • "Switch (LED OFF) → End"
