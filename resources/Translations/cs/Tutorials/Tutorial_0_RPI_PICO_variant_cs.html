Jak nastavit Raspberry Pi Pico pro použití s OmniBoard Studio

V tomto tutoriálu vás provedeme nastavením Raspberry Pi Pico pro použití s OmniBoard Studio.

Modely RPI Pico:

  • Raspberry Pi Pico
  • Raspberry Pi Pico W*
  • Raspberry Pi Pico WH*
  • Raspberry Pi Pico 2
  • Raspberry Pi Pico 2 W*
  • Raspberry Pi Pico 2 WH*

Budete potřebovat:

  • Kompatibilní desku Raspberry Pi Pico (viz seznam výše)
  • Micro USB kabel pro připojení Pico k vašemu počítači
  • Počítač s USB portem a přístupem k internetu

Kroky k nastavení vašeho Raspberry Pi Pico:

  • Stáhněte si nejnovější verzi MicroPython pro vaše RPI:
    • Pro Raspberry Pi Pico: https://micropython.org/download/RPI_PICO/
    • Pro Raspberry Pi Pico W a WH: https://micropython.org/download/RPI_PICO_W/
    • Pro Raspberry Pi Pico 2: https://micropython.org/download/RPI_PICO2/
    • Pro Raspberry Pi Pico 2 W a WH: https://micropython.org/download/RPI_PICO2_W/
  • Připojte váš Raspberry Pi Pico k počítači pomocí Micro USB kabelu, přičemž držte stisknuté tlačítko BOOTSEL (bílý tlačítko) na Pico. Tím se Pico přepne do režimu USB mass storage.
  • Zkopírujte stažený firmware MicroPython do kořenového adresáře Pico (Pico se zobrazí jako USB disk).
  • Jakmile je firmware zkopírován, Pico se automaticky restartuje a je připraveno k použití s OmniBoard Studio.
*W znamená, že tento model RPI má zabudovanou WiFi.
*WH znamená, že tento model RPI má zabudovanou WiFi a předpájené piny (headery).
