Jak nastavit linuxové Raspberry Pi pro použití s OmniBoard Studio

V tomto tutoriálu vás provedeme nastavením linuxového Raspberry Pi pro použití s OmniBoard Studio.

Modely RPI:

  • Raspberry Pi 5
  • Raspberry Pi 4 Model B
  • Raspberry Pi 3 Model B+
  • Raspberry Pi 3 Model B
  • Raspberry Pi 3 Model A+
  • Raspberry Pi 2 Model B
  • Raspberry Pi 1 Model B+
  • Raspberry Pi 1 Model B
  • Raspberry Pi 1 Model A+
  • Raspberry Pi 1 Model A
  • Raspberry Pi Zero 2 W*
  • Raspberry Pi Zero 2 WH*
  • Raspberry Pi Zero W*
  • Raspberry Pi Zero

Budete potřebovat:

  • Kompatibilní desku Raspberry Pi (viz seznam výše)
  • MicroSD kartu, minimálně 8 GB, ale doporučuje se 32 GB+ Class 10
  • Napájecí zdroj pro vaše Raspberry Pi (postačí jakýkoliv běžný 5V USB napájecí zdroj)
  • HDMI kabel a monitor (volitelné, pro počáteční nastavení). U některých modelů budete potřebovat redukci na micro HDMI.

Kroky k nastavení vašeho Raspberry Pi:

  1. Stáhněte si nejnovější verzi aplikace Raspberry Pi Imager pro snazší nastavení. Odkaz: https://www.raspberrypi.com/software/.
  2. Vložte MicroSD kartu do počítače a spusťte Raspberry Pi Imager.
  3. Vyberte váš model RPI a požadovaný OS (doporučeno: Raspberry Pi OS 64-bit. Můžete vybrat verzi lite, ale přijdete o možnost používat grafické uživatelské rozhraní (GUI) vašeho RPI, což není pro začátečníky příliš přátelské).
  4. Vyberte vaši MicroSD kartu jako cílové úložiště.
  5. Klikněte na "Další" a zadejte požadovaný Hostname (Název vašeho RPI), Username (přihlašovací uživatelské jméno) a Password (přihlašovací heslo). Nezapomeňte si své uživatelské jméno a heslo zapamatovat!
  6. Zadejte název a heslo vaší WiFi sítě. Nové sítě můžete přidat později v nastavení RPI přes GUI nebo SSH.
  7. Vyberte časové pásmo a rozložení klávesnice.
  8. Povolte SSH a VNC, pokud chcete využívat funkce vzdáleného přístupu.
  9. Klikněte na "Zapsat" (Write) pro zahájení zápisu obrazu na MicroSD kartu. POZNÁMKA: Tímto vymažete všechna data na MicroSD kartě!
  10. Po dokončení zápisu bezpečně odeberte MicroSD kartu z počítače a vložte ji do vašeho Raspberry Pi.
  11. Zapněte své Raspberry Pi a připojte klávesnici a myš (pokud používáte GUI).
  12. Prvotní konfigurace a připojení
    • Pokud používáte GUI, zadejte své uživatelské jméno a heslo pro přihlášení. Poté otevřete Terminál (Aplikace > Příslušenství > Terminál).
    • Pokud používáte SSH, najděte IP adresu vašeho RPI v administrátorském panelu vašeho routeru nebo pomocí nástroje jako nmap, a poté se připojte pomocí SSH klienta (např. PuTTY ve Windows, Terminál v macOS/Linuxu). Příkaz: `ssh username@ip_address`. Poté napište yes a zadejte své heslo. Nyní se nacházíte v příkazovém řádku RPI.
    • Spusťte příkazy pro aktualizaci vašeho systému: `sudo apt update && sudo apt full-upgrade -y`
    • Pokud používáte GUI, můžete použít RealVNC Viewer k připojení k desktopovému prostředí vašeho RPI. POZNÁMKA: V nastavení RPI musíte povolit VNC (Předvolby > Konfigurace Raspberry Pi > VNC).
    • Restartujte své Raspberry Pi, abyste aplikovali všechny změny, pomocí `sudo reboot`.

Červená LED dioda indikuje napájení a zelená LED dioda indikuje čtení/zápis na MicroSD kartu.

*W znamená, že tento model RPI má zabudovanou WiFi. Všechna RPI od modelu 3 výše mají zabudovanou WiFi.
*WH znamená, že tento model RPI má zabudovanou WiFi a předpájené piny (headery).
